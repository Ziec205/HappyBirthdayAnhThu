# 🎂 Birthday Gift Website

Một trang web sinh nhật đặc biệt với giao diện màu xanh dương tuyệt đẹp.

## 🌟 Tính năng

- ✨ Thiết kế hiện đại với màu xanh dương chủ đạo
- 🎈 Hiệu ứng bóng bay bay lên liên tục
- 🎊 Hiệu ứng pháo giấy khi nhấn nút
- 💙 Animations mượt mà và sinh động
- 📱 Responsive, hoạt động tốt trên mọi thiết bị
- 💌 Tin nhắn đặc biệt ẩn đợi người nhận khám phá

## 🚀 Hướng dẫn deploy lên GitHub Pages

### Bước 1: Tạo repository trên GitHub

1. Đăng nhập vào [GitHub](https://github.com)
2. Nhấn nút **"New"** để tạo repository mới
3. Đặt tên repository (ví dụ: `birthday-gift`)
4. Chọn **Public** (để có thể dùng GitHub Pages miễn phí)
5. Nhấn **"Create repository"**

### Bước 2: Upload code lên GitHub

Mở Terminal/PowerShell trong thư mục này và chạy:

```bash
git init
git add .
git commit -m "Initial commit - Birthday website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/birthday-gift.git
git push -u origin main
```

*Thay `YOUR_USERNAME` bằng tên GitHub của bạn*

### Bước 3: Kích hoạt GitHub Pages

1. Vào repository trên GitHub
2. Nhấn vào tab **Settings**
3. Chọn **Pages** ở menu bên trái
4. Trong phần **Source**, chọn **main branch**
5. Nhấn **Save**

### Bước 4: Truy cập website

Sau vài phút, website sẽ có sẵn tại:
```
https://YOUR_USERNAME.github.io/birthday-gift/
```

## 🎨 Tùy chỉnh

Bạn có thể chỉnh sửa file `index.html` để:
- Thay đổi nội dung lời chúc
- Đổi màu sắc theo ý thích
- Thêm hình ảnh cá nhân
- Điều chỉnh hiệu ứng

## 💡 Tips

- Gửi link cho người nhận vào đúng ngày sinh nhật để tạo bất ngờ!
- Có thể chia sẻ qua SMS, email, hoặc mạng xã hội
- Website hoạt động offline, có thể tải về điện thoại để xem
